这是一篇关于 Docker 容器技术的笔记。Docker 是一个开源的应用容器引擎，让开发者可以打包他们的应用以及依赖包到一个可移植的容器中，然后发布到任何流行的 Linux 或 Windows 机器上。

Docker 的核心优势包括：
1. 轻量级虚拟化
2. 快速部署
3. 环境一致性
4. 资源隔离

通过 Dockerfile 可以定义镜像构建过程，使用 docker-compose 可以管理多容器应用。
