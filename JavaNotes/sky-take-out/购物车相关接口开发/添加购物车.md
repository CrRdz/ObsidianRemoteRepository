# 需求分析和设计

## 接口设计
- 基本信息
	- Method：POST
	- Path：/user/shoppingCart/add
- 请求参数：
	Body

| 参数名称      | 类型      | 是否必须 | 默认值 | 备注   |
| --------- | ------- | ---- | --- | ---- |
| setmealId | integer | 否    |     | 套餐id |
| dishId    | integer | 否    |     | 菜品id |
| flavor    | string  | 否    |     | 口味   |
- 返回数据 

| 名称   | 类型      | 是否必须 | 默认值 | 备注  | 其他信息         |
| ---- | ------- | ---- | --- | --- | ------------ |
| code | integer | 必须   |     |     | format：int32 |
| data | string  | 非必须  |     |     |              |
| msg  | string  | 非必须  |     |     |              |
